📘 Introduction

Welcome to the SentiScan API and platform documentation. Here you'll learn how to integrate our sentiment and emotion analysis tools into your app or workflow.

🔐 Authentication

Each user is provided an api_key after registration. This key must be sent in requests to access the API securely.

Example usage:

{ "api_key": "your-api-key-here", "text": "I'm happy today!" }

🧠 API Usage

Send a POST request to /analyze with your API key and input text.

The response will contain sentiment scores, keywords, emotion label, and more.

📊 Usage Limits

Each user has a free tier limit of 20 analyses per day.
